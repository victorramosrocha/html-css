Sobre o HTML

HTML não é uma linguagem de programação mas uma linguagem de marcação ou linguagem de desenvolvimento.

  1. Foco na Estrutura e Conteúdo:

    • O HTML se concentra na estrutura e no conteúdo da página, definindo elementos como títulos, parágrafos, imagens e links.
    • Ele não possui recursos para executar lógicas complexas ou realizar cálculos, características presentes em linguagens de programação.
