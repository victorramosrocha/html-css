Versionamento

Porque versionar o site?

Para criar sites extensos, é prudente usar o versionamento. Caso seja feita alguma alteração no site que precise ser desfeita, o versionamento será útil tanto para a segurança quanto para a agilidade.

O que é preciso?

No entanto, para fazer esse versionamento é necessário um software muito importante, que é o gerenciador de versões. E um software livre e gratuito é o git.

Respositórios

O que é um repositório?

Repositório é um local onde guardamos coisas.

Local

Em um repositório local armazenamos nosso backup em nossa propria máquina. Usando por exemplo, o aplicativo git que é livre e gratuito.

Remoto

Já em um repositório remoto armazenamos o nosso backup em um servido. Pode ser usado por exemplo o GitHub que é repositório remoto gratuito.

Push

O push é quando você envia seu backup de um repositório local para um remoto ou vice-versa.
