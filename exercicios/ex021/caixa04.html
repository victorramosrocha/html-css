Teste de borda

Para colocar bordas é simples. Basta criar uma imagem num edito como o gimp ou buscar da internet e utilizar o css da seguinte maneira, para criar uma borda em volta de um <h1> por exemplo:

                h1{

                    border: 10px solid black;

                    padding: 10px;

                    border-image-source: url('imgbolas.png');

                    border-image-slice: 27;

                    border-image-repeat: repeat;

                }

ou ainda uma shorthand

            h1{

                border: 10px solid black;

                padding: 10px;

                border-image: url('imgbolas.png') 27    repeat 

            }

Não é muito agradável mas pode ser necessário. E lembrando ainda que nem todos os navegadores são compativeis com esse tipo de borda.
