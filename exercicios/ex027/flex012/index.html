Aula 10 - A Propriedade Flex

Nas aulas anteriores vimos as prorpiedades de flex: flex-basis, flex-shrink, flex-grow. Nesta aula seguiremos a recomendação da W3School, que aconselha utilizar a Propriedade flex.

Sem utilizar diretamente flex

    
    flex-basis: 150px;
    flex-grow: 0;
    flex-shrink: 1;

Utilizando somente o flex


    flex: 0 1 150px;

Então, temos flex como: flex: 'grow', 'shrink', 'basis';

Mas, é possível simplificar ainda mais? Claro que sim, pequeno gafanhoto!

  • flex: 0 1 auto; / flex: initial; - comportamento padrão dos itens flexíveis.
  • flex: 0 0 auto; / flex: none; - não permite os itens aumentarem ou diminuirem.
  • flex: 1 1 auto; / flex: auto;- é o máximo de flexibilidade de um item, limitado apenas por seu conteúdo.
  • flex: 3; / flex: 3 1 auto; - quando usado um unico valor "flex: 'n';" apenas o grow é alterado. O shrink e basis permanecem padrão (shrink = 1 e basis = auto).
