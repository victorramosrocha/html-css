Anatomia dos Itens Flexiveis

Order by

No CSS3, a propriedade order é utilizada para definir a ordem de exibição dos itens dentro de um elemento pai com display: flex.

No exemplo abaixo, temos cinco caixas numeradas de 1 a 5, cuja ordem natural de exibição seria crescente. No entanto, é possível alterar essa ordem visualmente utilizando a propriedade order no CSS.

Para isso, basta referenciar cada item pelo seu id e atribuir um valor numérico à propriedade order, conforme demonstrado a seguir: 

                
                    div#box-1 {
                        order: 3;
                    }

                    div#box-2 {
                        order: 1;
                    }

                    div#box-3 {
                        order: -1;
                    }

                    div#box-4 {
                        order: 2;
                    }

                    div#box-5 {
                        order: 0;
                    }

Vale lembrar que valores negativos também podem ser utilizados na propriedade order, seguindo a ordem numérica crescente (por exemplo: -2, -1, 0, 1, 2...).

Além disso, os valores atribuídos não precisam seguir uma sequência contínua. Ou seja, não é necessário utilizar números de forma ordenada (1, 2, 3...). É possível utilizar qualquer valor inteiro, como no exemplo abaixo: 

                
                    div#box-1 {
                        order: -1;
                    }

                    div#box-2 {
                        order: 0;
                    }

                    div#box-3 {
                        order: 5;
                    }

                    div#box-4 {
                        order: -2;
                    }

                    div#box-5 {
                        order: 7;
                    }
