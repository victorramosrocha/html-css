Essas são algumas das principais fomatações de textos

Negrito / Destaque

Nesta frase temos um termo em Negrito usando a tag <b> de "Bold" (tag não semântica, "apenas forma, sem sentido").

Nesta outra frase temos um termo em Destaque usando a tag <strong> de "Destaque" (tag semântica "com sentido").

O Bold me diz que o termo está em Negrito, já o strong me diz que está em Destaque.

Itálico / Ênfase

O mesmo ocorre com o termo em itálico usando a tag <i> (tag não semântica).

Ainda podemos usar o termo em itálico com a tag <em> (tag semântica).


A tags de formas estão sendo substituidas por tags semânticas no html, ou seja, tags com sentido.

Texto Marcado

Criamos um texto marcado usando a tag <mark>.

Podemos usar a tag <mark style="background-color: 'Nome ou código da cor'"> para definir a cor de um marca texto pontual.

Mas podemos usar também a tag <style> dentro do head para definir a cor de todos os marca textos.

Essa tag <style> não é HTML, já passa a ser CSS.

Texto grande / Texto pequeno

Criando um texto grande e um texto pequeno usando as tags <big> e <small>.

A tag <big> está obsoleta enquanto a <small> continua sendo utilizada.

Texto deletado

Um texto que pode ser lido, mas deve ser desconsiderado pode usar a tag <del>.

Texto inserido

Já um texto inserido ou sublinhado deve usar a tag <ins> que substitui a tag <u> que ficou obsoleta.

Texto sobrescrito

Para escrever o termo 'X20 + 3' por exemplo, escrevemos o 20 entre a tag <sup> para elevar apenas o texto 20.

Texto subscrito

Para o termo 'H2O', escrevemos o 2 entre a tag <sub>.
